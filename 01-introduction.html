Installing Git

در این بخش یاد می‌گیریم چگونه Git را روی سیستم نصب کنیم، نصب را بررسی کنیم و با محیط‌های اجرای دستورات Git آشنا شویم.

پیش‌نیازها

  • دسترسی به اینترنت
  • دسترسی نصب نرم‌افزار روی سیستم (Administrator)
  • یکی از سیستم‌عامل‌های Windows، macOS یا Linux

دانلود Git

Git فقط باید از سایت رسمی دانلود شود.
مورد توضیح
وب‌سایت رسمی https://git-scm.com
دلیل استفاده امن‌ترین و به‌روزترین نسخه

نصب Git روی Windows

1. دانلود

  1. سایت https://git-scm.com را باز کنید
  2. روی گزینه Download for Windows کلیک کنید
  3. فایل نصب (.exe) دانلود می‌شود

2. اجرای فایل نصب

  • روی فایل دانلود شده دوبار کلیک کنید
  • پنجره نصب باز می‌شود

3. مراحل نصب

بخش تنظیم پیشنهادی

نصب Git روی macOS

روش 1 — استفاده از Homebrew (پیشنهادی)

brew install git

روش 2 — دانلود مستقیم

  • به سایت رسمی Git مراجعه کنید
  • نسخه macOS را دانلود و نصب کنید

نصب Git روی Linux

Ubuntu / Debian

sudo apt update
sudo apt install git

Fedora

sudo dnf install git

Arch Linux

sudo pacman -S git

بررسی نصب Git

برای بررسی نصب، دستور زیر را اجرا کنید:

git --version

اگر خروجی مشابه زیر نمایش داده شد، Git به‌درستی نصب شده است: 

git version 2.xx.x

تنظیم اولیه Git

پس از نصب باید اطلاعات کاربر تنظیم شود:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

محیط‌های اجرای دستورات Git

Git یک ابزار است، نه یک محیط

Git خودش یک محیط برای تایپ دستور ندارد.

Git یک ابزار خط فرمان است و برای استفاده از آن باید از ترمینال استفاده شود.

  • Git = ابزار
  • Terminal = محیط اجرای دستور

ترمینال چیست؟

ترمینال محیطی متنی است که در آن می‌توان دستورات سیستمی و ابزارهایی مثل Git را اجرا کرد.

Git Bash چیست؟

  • همراه Git در ویندوز نصب می‌شود
  • محیطی شبیه لینوکس دارد
  • برای یادگیری Git بسیار مناسب است
  • ساده‌ترین گزینه برای کاربران ویندوز

Command Prompt (CMD) چیست؟

ترمینال پیش‌فرض ویندوز است و در صورت نصب صحیح Git می‌تواند دستورات را اجرا کند.

PowerShell چیست؟

ترمینال پیشرفته‌تر ویندوز که قابلیت اجرای دستورات Git را دارد.
